# Microservices Architecture Requirements

In architecting this software, including the new EV charging capability, you should propose a resultant microservices-based architecture to support the overall application.

**Note:** You are asked to *describe* the microservice-based software architecture, but you are *not required* to develop the extended microservices software implementation at this stage.

## Design a Preliminary Microservices Architecture:
- Identify services (align with bounded contexts)
- Identify key responsibilities of each service
- Describe APIs/endpoints (external facing and service-to-service endpoints)
- Identify separate DBs per service

## Documentation
Provide a written portion describing the microservices-based architecture. Include:
- A proposed microservices architecture diagram including services, APIs/endpoints and per service DBs.
